    MySQL的基本操作

    • 默认信息 

      默认端口：3306
默认用户：root
默认字符集：latin1(应设置为utf-8或gbk)

    • mysql服务的启动，登录，退出及停止

    • 启动服务
    net start mysql
    • 登录mysql
    mysql -h (填入host,本地填写127.0.0.1) -u root -p
mysql -h 127.0.0.1 -u root (库名) -p 登入指定的数据库
    • 退出
    exit & quit
    • 停止服务
    net stop mysql

    • MySQL的数据类型

      • 整型

        smallint(2)
        int(4)
        integer同int
        bigint(8)

      • 浮点型

        float(4)
        double(8)

      • 字符串(字符)型

        char(固定长度字符串)
        varchar(可变长度字符串)

      • 大对象

        text 非二进制大对象
        blob 二进制大对象

      • 日期时间

        date:YYYY-MM-DD
        datetime:YYYY-MM-DD HH:MM:SS
        timwstamp:YYYY-MM-DD HH:MM:SS
        time:HH:MM:SS
        year:YYYY

    • SQL语句语法

      • 数据定义语言(eg: create,drop,alter)
        create database school; /*创建出具库school*/

    showdatabases; /*显示所有数据库，等于：

    select database()*/

    use school; /*指定默认数据库*/

    create table student(       /*创建表*/
        id int(10) primary key auto_increment,  /*设置主键自增*/
        name varchar(8),
        sex char(1),
        age int(2),
        score float(6,2)
    );

    show tables; /*显示当前库的表清单*/

    show columns from student; /* 显示指定表结构，等同describe student;*/

    show create table student;  /*显示建表sql语句（信息）*/

    drop table student; /*删除指定表*/

    drop database school;   /*删除数据库*/


>* 数据查询语言(eg:__select__)

    select * from student;

    select * from student where id<5;

    select count(*) from student;   /*查询表中数据量*/

    select max(score) from student; /*查询指定量最大值*/

    select name,id,score from student order by score desc;  /*按分数降序查询指定信息*/

    select * from student where name like "Li%";    /*近似查找*/


>* 数据操作语言(eg:__insert,delete,update__)


    insert into student values(null,"LiMing","m",19,99,5)  /*数据插入*/

    select * from student;    /*查询当前表中所有数据*/

    alter table student modify sex char(2); /*修改sex的数据长度*/

    update student set sex="男" where id=3; /*数据更新*/

    delete from student where name="王五"; /*数据删除*/

    delete from student; /*数据清空*/


>* 数据控制语言(eg:__grant,revoke,commit,rollback__)

>* 作用范围      
    数据操作语言针对表中数据，数据定义语言针对数据库或表
    • 注意
      • 主键自增（不使用序列，通过 auto_increment）
      • 所有语句都要在结尾加上分号表示结束
