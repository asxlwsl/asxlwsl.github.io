Markdown基础学习

1. MarkDown中的语法规则

  • MarkDown的标题
    • 使用[^============和———]:”=”为一级标题，“-”为二级标题
    • 使用“#”标记
  • MarkDown的段落
    • 段落的换行使用两个以上的空格加回车(未生效)或者空一行
  • MarkDown字体
    斜体
    斜体
    粗体
    粗体
    粗斜体
    粗斜体
  • 分割线
  • 有以下几种方式（字符大于等于三）
    • - - -
    • ————
  • 删除线
    • 在要删除的文字两端加上两个波浪线
      ~~jk~~
  • 下划线
    • 通过HTML的u标签实现
  • 脚注
    • 这是要注明的文字。
      ^[注释]
  • 列表
    • 无序列表用 * 、+ 、-
    • 有序列表 数字加点
    • 列表嵌套 子列表前加四个空格
  • 区块

列表内使用区块加三个空格

1

2

  • MarkDown代码
    • 如果是段落上的一个函数或片段的代码可以用反引号将他包起来
    • 代码区块使用4空格或一个制表符
    • 也可以用‘‘‘内容‘‘‘包裹并指定一种语言（也可以不指定）
<?php
        echo '123';
        function test(){
            echo 'test'
        }

    alert('Hello');
});

  • MarkDown链接
    • 链接名称+链接地址 [name]+(url)
      baidu
    • 直接使用
      https:www.bilibili.com
    • 高级链接
        可以用变量代替网址,文档末尾附带变量地址
      Baidu
